    <div class="overview-block text-accent-left full-width-block">
      <h3 class="overview-heading">Topics Covered</h3>
      <ul class="styled-curriculum-list">
        <li>
          <div class="list-text"><strong>Consequences of $\mathsf{AD}$</strong> — $\aleph_1$ is measurable, every subset of $\mathbb{R}$ is Lebesgue measurable.</div>
        </li>
        <li>
          <div class="list-text"><strong>Descriptive Set Theory</strong> — $\Pi_1^1$ normal form, Mostowski absoluteness theorem, Shoenfield absoluteness theorem.</div>
        </li>
        <li>
          <div class="list-text"><strong>Large Cardinals & Determinacy</strong> — $0^\sharp$, elementary embeddings, analytic determinacy.</div>
        </li>
        <li>
          <div class="list-text"><strong>Admissible Set Theory</strong> — Kripke-Platek set theory, $\omega$-models, well-founded models, admissible ordinals.</div>
        </li>
      </ul>
    </div>
